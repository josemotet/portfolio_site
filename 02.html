Grundlæggende Web

Opgave

I temaet ”Grundlæggende web” blev vi introduceret for HTML, CSS, Grid, Flex og Media Querires. Dette skulle bruges til at få opbygget vores responsive site ”Min Computer” til studiestartsprøven. Nedenfor kan man se de forskellige værktøjer vi har arbejdet med for at kunne bruge HTML og CSS optimalt. Samt link til studiestartsprøven.

Link til kodet site

Grid

Grid består af en grid container (parent) samt grid items (child). Grid hjælper med at opstille en struktureret og responsiv sideopbygning. I studiestartsprøven jeg blandt andet benyttet det til at opstille billeder på mit site. Dette kan ses på billederne nedenfor hvor jeg ved hjælp af HTML og CSS har positioneret de forskellige billeder.

Flex

Ved at bruge Flex kan man kontrollere rumfordeling, justering og rækkefølge af elementer. Dette har jeg gjort i min menu i studiestartsprøven. Dette gjorde det dermed overskueligt for brugeren at navigere rundt i de forskellige menuer.

Mediaquery

Ved at bruge mediaquery kan man skabe responsive løsninger, der tilpasser sig skærmstørrelser. Sammen med Flex og Grid kan man dermed muligøre en dynamisk omorganisering af layoutet ved at justere rundt. Dette giver en optimal visning på forskellige enheder. I studiestartsprøven havde jeg sat mediaqueryen i min layout CSS på min-width 800px, som dermed tilpassede mit layout. Dette kan ses nedenfor på en af siderne, hvor opstillingen af elementer rykker sig efter om det er på mobil eller desktop.
